Getting back into Subnautica

I first started playing Subnautica back when it released early access in 2014. Though the game had a long way to go I knew it had protental.

Later when Subnautica reached full release I booted it up again. I played it off and on though the years but I never finished it.

Now it is 2024 and the game is quite mature now. It has received a healthy amount of updates over the years. Unfortunately when I booted it up this most recent time my saved game files were gone. My guess is that I last played the game so many updates ago that my old saved files lost compatibility with current builds of the game over the years. This was sad because I had a special saved file from a time I was playing the game with someone special. Despite this I am having a lot of fun with the new save I recently started.
